# Koncepthive Fitness

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React
- Radix UI primitives (shadcn-style components)

## Setup Instructions

1. Clone the repository:

	```bash
	git clone <your-repo-url>
	cd Koncepthive_Fitness
	```

2. Install dependencies:

	```bash
	npm install
	```

3. Run the development server:

	```bash
	npm run dev
	```

4. Open in browser:

	```
	http://localhost:3000
	```

5. Production build (optional):

	```bash
	npm run build
	npm run start
	```
